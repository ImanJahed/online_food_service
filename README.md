# نحوه اجرای پروژه

1- ساخت محیط مجازی با دستور:

`python -m venv venv`

2-نصب نیازمندهای پروژه

`pip install -r requirements.txt`

3- ساخت دیتابیس

`python manage.py migrate`

4- اجرا پروژه

`python manage.py runserver`
